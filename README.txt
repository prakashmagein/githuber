== Description ==

Githuber is a github-style WordPress theme with preferred features for developers.

For more information about Githuber please go to https://github.com/terrylinooo/githuber .

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Githuber in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
